# PRISMA — Gerenciador de mídia pra quem edita

![status](https://img.shields.io/badge/status-À_VENDA-brightgreen) &nbsp; por **Paulo Adriel** · [Paulocodex](https://paulocodex.com)

PRISMA é o 'Eagle que entende vídeo': um gerenciador de assets desktop, não-destrutivo, que indexa milhares de arquivos e devolve qualquer um em segundos — por cor, nome, tag ou conteúdo. Prepara a mídia pro corte (CST DaVinci, proxies, conserto de codec), roda IA local e baixa de YouTube/Spotify/Epidemic. Núcleo grátis pra sempre; Pro vitalício.

## O que faz

- Busca instantânea por cor, nome e conteúdo
- IA local: auto-tag, upscale, remover fundo
- CST DaVinci + proxies + conserto de codec
- Baixar de YouTube / Spotify / Epidemic
- Marca d'água, GIF, contact-sheet, OCR
- Não-destrutivo: nunca toca nos originais
